# Platform Layer

This directory is reserved for **platform-specific** adapters and notes.

Boundary:

- `skills-core/` is the only source of project LibRPA skills.
- `platform/` must not contain duplicated domain skills.
- `platform/` may contain launcher notes, manifests, optional wrappers, and per-platform integration guidance.

Targets:

- `platform/openclaw/`
- `platform/opencode/`
- `platform/codex/`
